FilmX Composite nodes

Easily apply a certain look to your footage or image with a single node group that can be easily adjusted and customized. Each film (node group) has a Letter and number to identify it. The letter is the type of the film and the number is the type number of the film. E.G is a node group has the name "C5" it is a C Type film and is the 5 film in the C type. FilmX is easy to install and use, just download the file and place the blender file where you store your assets. Then you can append the desired node group film to your project and start using it. FilmX is also entirely open source and free to use under the public domain meaning you can use it in any project and modify it to your liking.
