OpenBlender

A collection of assets, demo files, experiments and templates for blender that are all licensed under the public domain for anyone to use and modify.

Objects

These 3d objects and models are ready to be used in any project and come in a variety of formats

General Demo Files

These are demo files for testing and benchmarking blender

Playgrounds

These files are blank scenes with a few objects and lights ready for experiments and physics simulations.

FX

These files are quick and simple to use effects for any project that can be added to any scene using the append function or importing as a video

Compositing

Compositing experiments and demo files that can be used as a starting point for any project
